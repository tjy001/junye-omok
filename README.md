## Installation 
### run `npm install` or `npm i`

## Running the app
### run `npm start`
The app should be started on `localhost:3000` by default

## Creating a game session
1. Enter a username in the username textbox
2. Click on `Create a Game`
3. A 6-digit game id will be created and displayed
4. The 6-digit game id will be used by another player to join the game seesion you've created

## Joining a game session
1. Enter a username in the username textbox
2. Enter a valid 6-digit game id (from the other player creating a game session)
3. Click on `Join a Game`
